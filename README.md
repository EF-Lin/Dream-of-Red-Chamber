# 紅樓夢（Dream of Red Chamber）電子書
## 簡介
《紅樓夢》，又名《石頭記》，作者曹雪芹，中國四大小說之一，章回小說的巔峰之作，書中集傳統文化大成，呈現出中國古代封建社會的百般樣貌，細膩描寫貴族大家的興衰迭起與悲歡離合，可說是中國文學史上最重要的藝術品與著作。

現網際網路中的《紅樓夢》線上資源主要來自[維基文庫](https://zh.wikisource.org/wiki/%E7%B4%85%E6%A8%93%E5%A4%A2_(%E6%B6%88%E6%AD%A7%E7%BE%A9))與[中國哲學書電子化計劃](https://ctext.org/hongloumeng/zh)等網站，但其皆為網頁形式，並非PDF檔案，且現存的[PDF檔案](https://archive.org/details/20201227_20201227_2155/page/n5/mode/2up)也存在大量錯字，因此編者萌生編寫《紅樓夢》電子書的想法，進一步催生出本專案。

本專案旨在蒐集線上資源，編寫一部《紅樓夢》電子書。同時遵循一項原則：忠於文本，電子書內容將盡可能減少對於文本的解釋與揣測，忠實的呈現《紅樓夢》一書。

本專案以LaTex編寫而成，《紅樓夢》文本內容來自維基文庫的[脂硯齋重評石頭記](https://zh.wikisource.org/wiki/%E8%84%82%E7%A1%AF%E9%BD%8B%E9%87%8D%E8%A9%95%E7%9F%B3%E9%A0%AD%E8%A8%98)，按語同樣來自維基文庫，少部分為編者自行補充，惟編者才疏學淺，對於《紅樓夢》之相關歷史與研究所知甚少，還望各位不吝補正。

## 版本
《紅樓夢》現存版本眾多，經過學者考證後認為手抄本「庚辰本」應為最貼近曹雪芹原意的版本，可惜僅存前八十回，後續內容已佚失不可考，本專案也以庚辰本為底。同時為了達到忠於原著的目的，本專案並未納入程偉元、高鶚等人編撰的程甲本與程乙本。更多資訊可參考[想讀《紅樓夢》，買哪一個版本好？](https://vocus.cc/article/64b25c7dfd89780001387f56)與[紅樓夢抄本列表](https://zh.wikipedia.org/zh-tw/%E7%BA%A2%E6%A5%BC%E6%A2%A6%E6%8A%84%E6%9C%AC%E5%88%97%E8%A1%A8)。

### 脂硯齋重評石頭記
脂硯齋為清朝文人，其留下的庚辰本、己卯本、甲戌本等手抄本與批語（脂批）為後世研究紅樓夢留下一盞明燈，脂批可理解為脂硯齋在閱讀《紅樓夢》時的筆記，其不僅多次透露佚失部分劇情，更指點後人該如何理解《紅樓夢》的文字。

根據維基文庫的介紹：
>...匯集了《脂硯齋重評石頭記》庚辰本、己卯本、甲戌本的正文和批語。三本所缺六十四回、六十七回和二十二回未完成部分，以《紅樓夢》的其它脂本系統的抄本的配入；六十四回的文字主要來自俄藏本，六十七回和二十二回未完成部分主要參照戚序本。

在此版本中，編者將脂批改以腳註（footnote）呈現，保證了閱讀時的連貫性。同時為了區別，按語將以`〈〉`的形式插入正文中。

### 紅樓夢庚辰本
此版本正文與《脂硯齋重評石頭記》相同，但刪除所有批語。並且將按語改以腳註方式插入。

## 修正
編者針對文本進行些許修正，替換掉部分異體字，主要目的是確保LaTex編譯過程正常。以下列出修正文字。

- 作「爲」為「為」
- 作「幷」為「并」
- 作「册」為「冊」
- 作「兎」為「兔」
- 作「别」為「別」

並且編者也刪除維基文庫中部分對於文本解釋的按語，避免讀者閱讀時受到影響。

## 編譯
除了至[Releases](https://github.com/EF-Lin/Dream-of-Red-Chamber/releases)中下載編譯後的PDF檔外，讀者也可以自行編譯LaTex檔案。但需要在電腦中先行安裝LaTex編譯相關軟體，在此不另行贅述。

### 指令
```bash
git clone https://github.com/EF-Lin/Dream-of-Red-Chamber.git
cd 紅樓夢庚辰本 or cd 硯齋重評石頭記
xelatex main.tex
xelatex main.tex
```
編譯兩次是為了讓目錄正常顯示。
